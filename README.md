ledcheck
===============================

version number: 0.0.1
author: Emmet Tracey

Overview
--------

Counts the number of LED lights that are on

Installation / Usage
--------------------

To install use pip:

    $ pip install ledcheck


Or clone the repo:

    $ git clone https://github.com/Emmet62/ledcheck.git
    $ python setup.py install
